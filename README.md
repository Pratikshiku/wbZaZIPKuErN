**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告信息
5. 留言
6. 个人中心
7. 修改密码
8. 个人信息
9. 公告信息管理
10. 缴费
11. 留言
12. 维修管理
13. 行程轨迹管理
14. 轮播图信息


**管理员功能：**
